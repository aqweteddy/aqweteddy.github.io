安裝 Hugo 建制部落格，並集成 LaTeX

  • 安裝 Hugo
  • Hugo 基本指令
  • 增加 LaTeX

Install Hugo

Environment

  • OS: MAC OS Mojave
  • Package Manager: brew

Step

  1. brew install hugo

Create new blog

  1. hugo new site [blog_name]

Create new article

  1. hugo new [path][page_name]
    • create .md in [path_to_project]/content

Run server

  • hugo server -D

github Page

  1. create a new empty project in github.
  2. use command hugo to save result in [proj]/public
  3. do following command:
cd public
git init
git add .
git commit -m "first"
git push

LaTeX
